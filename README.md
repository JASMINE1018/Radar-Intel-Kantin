# Radar-Intel-Kantin
Radar Intel Kantin adalah platform web yang mendigitalisasi pengelolaan kantin sekolah melalui sistem terintegrasi untuk Owner, Seller, dan User. Platform ini meningkatkan transparansi menu, harga, dan kualitas makanan, sekaligus mempermudah pengelolaan penjual serta pemantauan aktivitas kantin secara efisien.
